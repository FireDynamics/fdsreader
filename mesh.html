Meshes¶

class fds.mesh.Mesh(x_coordinates, y_coordinates, z_coordinates, label)[source]

3-dimensional Mesh of fixed, defined size. :ivar coordinates: Coordinate values for each of the 3 dimension. :ivar extent: Tuple with three tuples containing minimum and maximum coordinate value on the

corresponding dimension.

Variables

  • mesh –

  • n – Number of elements for each of the 3 dimensions.

  • n_size – Total number of blocks in this mesh.

  • label – Label associated with this mesh.

Parameters

  • x_coordinates – Coordinate values of x-axis.

  • y_coordinates – Coordinate values of y-axis.

  • z_coordinates – Coordinate values of z-axis.

  • label – Label associated with this mesh.

Creates a collection of slices by collecting all mesh (GRID) information in a given .smv file.

Parameters
