EvacCollection(evacs, base_path, times)

Collection of Evacuation objects.
class fdsreader.evac.EvacCollection.EvacCollection(evacs: Iterable[fdsreader.evac.evacuation.Evacuation], base_path: str, times: Iterable[float])[source]

Collection of Evacuation objects. Next to agent-class specific data (such as trajectories) lots of other data such as FED-data is provided via this class.

Variables

  • times – List of all time steps of the simulation.

  • z_offsets – The offset in z-direction for each mesh where the evac plane lays.

  • all_agents – Number of all agents per time step.

  • agents_inside_mesh – Number of all agents per time step inside a specific mesh.

  • number_of_deads – Number of dead agents per time step.

  • fed_max – FED max per time step.

  • fed_max_alive – FED max alive per time step.

  • exit_counters – Exit counts per time step.

  • target_exit_counters – Target exit counts per time step.

  • door_counters – Door counts per time step.

  • target_door_counters – Target door counts per time step.

Container to store evac data from evac simulations with FDS.
class fdsreader.evac.evacuation.Evacuation(class_name: str, quantities: List[fdsreader.utils.data.Quantity], color: Tuple[float, float, float])[source]

Container to store evac data from evac simulations with FDS.

Variables

  • class_name – Name of the evac class defined in the FDS input-file.

  • quantities – List of all quantities for which data has been written out.

  • color – Color assigned to the evac.

  • n_humans – Number of existing evacs for each timestep per mesh.

  • lower_bounds – Dictionary with lower bounds for each timestep with quantities as keys.

  • upper_bounds – Dictionary with upper bounds for each timestep with quantities as keys.
