Slice files¶

class fdsreader.slcf.slice.Slice(root_path: str, cell_centered: bool, multimesh_data: Collection[Dict])[source]
Slice file data container including metadata. Consists of multiple subslices, one for each

mesh the slice cuts through.

Variables

  • root_path – Path to the directory containing all slice files.

  • quantities – List with quantity objects containing information about the quantities calculated for this slice with the corresponding label and unit.

  • cell_centered – Indicates whether centered positioning for data is used.

  • times – Numpy array containing all times for which data has been recorded.

Part of a slice that cuts through a single mesh.

  • root_path – Path to the directory containing the slice file.

  • mesh – The mesh the subslice cuts through.

  • extent – Extent object containing 3-dimensional extent information. Values are indexes for the actual grid values of the mesh.

  • file_names – File names for the corresponding slice file depending on the quantity.
