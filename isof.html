Isosurface files¶

class fdsreader.isof.isosurface.Isosurface(root_path: str, double_quantity: bool, iso_filename: str, quantity: str, label: str, unit: str, viso_filename: str = '', v_quantity: str = '', v_label: str = '', v_unit: str = '')[source]
Isosurface file data container including metadata. Consists of a list of vertices forming a list

of triangles. Can optionally have additional color data for the surfaces.

Variables

  • file_path – Path to the binary data file.

  • v_file_path – Path to the binary data file containing color data.

  • quantity – Information about the quantity.

  • v_quantity – Information about the color quantity.

  • levels – All isosurface levels

  • n_vertices – The number of vertices for this isosurface.

  • n_triangles – The number of triangles for this isosurface.

  • _double_quantity – Defines whether there is color data for this isosurface or not.

  • _offset – Offset of the binary file to the end of the file header.

  • _v_offset – Offset of the binary file containing color data to the end of the file header.

property colors

Property to lazy load the color data that might be associated with the isosurfaces.

property has_color_data

Defines whether there is color data for this isosurface or not.

property surfaces

Property to lazy load a list that maps triangles to an isosurface for a specific level.

triangles()[source]

Property to lazy load all triangles of any level.

property vertices

Property to lazy load all vertices for all triangles of any level.
